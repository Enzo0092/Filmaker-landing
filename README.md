
# Landing Page - Filmmaker & Multimedia Editor

Landing page desarrollada para promocionar los servicios de un filmmaker y editor de contenido multimedial.
El proyecto está orientado a presentar servicios, portfolio, propuesta de valor y medios de contacto de forma moderna, visual y profesional.

## Descripción

Este sitio web funciona como una página de presentación para un profesional dedicado a la producción audiovisual, edición de video y creación de contenido para redes sociales, marcas, eventos y proyectos comerciales.

La landing busca transmitir una estética cinematográfica, moderna y premium, con foco en captar potenciales clientes y facilitar el contacto directo.

## Frameworks y herramientas

* **React:** librería utilizada para construir la interfaz mediante componentes reutilizables.
* **Vite:** herramienta de desarrollo utilizada para levantar el proyecto de forma rápida y optimizada.
* **TypeScript:** mejora la seguridad del código agregando tipado estático.
* **Tailwind CSS:** framework CSS utilizado para aplicar estilos de forma rápida y consistente.
* **Figma Make:** herramienta utilizada para generar una primera base visual del proyecto.
* **Git & GitHub:** control de versiones y colaboración en equipo.

## Instalación y ejecución local

Clonar el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

Ingresar a la carpeta del proyecto:

```bash
cd NOMBRE_DEL_PROYECTO
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el servidor de desarrollo:

```bash
npm run dev
```

Abrir en el navegador la URL que indique la terminal, normalmente:

```bash
http://localhost:5173
```
